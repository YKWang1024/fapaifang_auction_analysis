# 法拍房竞价策略分析

## 项目简介

本项目用于分析法拍房竞价记录，通过数据可视化和统计分析，帮助理解竞价节奏、竞买人行为模式，并从中总结成功策略和失败教训。

## 分析指标体系

### 竞买人维度分析

| 指标 | 说明 |
|------|------|
| 总竞价次数 | 该竞买人参与的总出价次数 |
| 初始竞价价格 | 该竞买人的首次出价金额 |
| 最高竞价价格 | 该竞买人的最高出价金额 |
| 相对全局时间差均值 | 每次出价与最近一次全局出价的时间间隔平均值 |
| 相对自己时间差均值 | 该竞买人每次出价与上一次出价的时间间隔平均值 |
| 相对全局金额差均值 | 每次出价与最近一次全局出价的金额差平均值 |
| 相对自己金额差均值 | 该竞买人每次出价与上一次出价的金额差平均值 |
| 震慑性加价次数 | 单次加价超过3%的次数 |

### 竞拍节奏维度分析

| 指标 | 说明 |
|------|------|
| 前期/中期/后期时间间隔 | 将竞拍分为三个阶段，分析各阶段平均响应时间 |
| 前期/中期/后期加价金额 | 各阶段平均加价幅度 |
| 节奏变化(时间) | 整体节奏是变快还是变慢 |
| 节奏变化(幅度) | 加价幅度是变大还是变小 |
| 最长/最短等待时间 | 全场最长和最短的出价间隔 |
| 最大单次加价 | 单次最大加价金额和百分比 |

## 使用方法

### 1. 竞买人维度分析

```bash
python analyze_bids.py
```

输出：各CSV文件对应的 `*_竞价分析结果.csv`

### 2. 竞拍节奏分析

```bash
python auction_rhythm_analyzer.py
```

输出：各CSV文件对应的 `*_节奏分析图.png`

## 图表说明

节奏分析图包含6个子图：

1. **价格与时间间隔变化趋势** - 按竞买人着色，显示价格走势和时间间隔
2. **各竞买人出价次数与最终出价** - 水平柱状图，展示各竞买人参与度
3. **每次加价幅度百分比** - 含3%震慑线参考
4. **节奏趋势（移动平均）** - 5次移动平均的时间间隔和加价金额
5. **竞拍阶段节奏对比** - 前期/中期/后期对比
6. **竞拍节奏统计摘要** - 关键指标汇总

## 关键发现

### 成功策略

- **高频次参与**：赢家竞价次数通常是其他竞争者的2-4倍
- **快速响应**：成功竞买人响应时间普遍较短（<100秒）
- **稳定加价**：采用小额稳定加价策略（0.3%-0.5%）

### 失败教训

- 单次出价后放弃
- 响应太慢，错失时机
- 过于激进的威慑性报价（>3%）效果有限

### 威慑性报价

数据表明，震慑性加价（单次超过3%）在所有案例中均未成功。建议谨慎使用。

## 数据格式

CSV文件列格式：
- 状态：出价状态
- 竞买号：竞买人标识
- 价格：出价金额
- 时间：出价时间（格式：YYYY年MM月DD日 HH:MM:SS）
